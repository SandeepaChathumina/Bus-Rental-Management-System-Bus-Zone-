# 🚌 BusZone+

BusZone+ is a **full-stack bus rental and management system** developed to simplify bus booking, scheduling, and fleet management. The system is designed to support real-world transportation needs with a user-friendly interface and efficient backend operations.

---

## 📌 Project Overview

BusZone+ provides a digital platform where users can easily rent buses while administrators manage buses, bookings, and schedules efficiently. This project was developed as part of an **academic project** to gain hands-on experience in building scalable web applications using the **MERN stack**.

---

## 🚀 Features

### 👤 User Features

* User registration and authentication
* Browse available buses
* Book buses based on availability
* View booking history

### 🛠️ Admin Features

* Add, update, and remove bus details
* Manage bookings and schedules
* View user and booking information
* Dashboard-style management interface

## ⚙️ Installation & Setup

### Prerequisites

* Node.js
* MongoDB
* Git

### Steps

1. **Clone the repository**

```bash
git clone https://github.com/YOUR_USERNAME/BusZone+.git
```

2. **Navigate to the project directory**

```bash
cd BusZone+
```

3. **Install backend dependencies**

```bash
cd backend
npm install
```

4. **Install frontend dependencies**

```bash
cd ../frontend
npm install
```

5. **Run the backend server**

```bash
cd backend
npm start
```

6. **Run the frontend application**

```bash
cd frontend
npm start
```
